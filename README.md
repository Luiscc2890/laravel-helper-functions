Provides Laravel-specific and pure PHP helper functions.

## Dependencies
- `PHP >=5.4.0`
- `Laravel >=5.2`

## Usage

1. Install package through `composer`:
    ```shell
    composer require illuminated/helper-functions
    ```

2. That's it! Now you can use any of provided helper functions.
